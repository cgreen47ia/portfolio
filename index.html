About

Learning by building, one project at a time.

I'm a vibe coding student exploring how far curiosity and modern tools can take an idea — from rough concept to working prototype.

Each project in this portfolio started with a simple question: what if I built something useful, learned along the way, and let design guide the result? Bold Canvas, my design system, keeps every interface consistent and intentional.

When I'm not coding, I'm sketching layouts, refining typography, and looking for the next problem worth solving.
